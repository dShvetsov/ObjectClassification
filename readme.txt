3. Пирамида дескрипторов реализуется в функции pyramid 
класс HOG, принимает на вход ссылку на изображение, метод ComputeHog(n), высчитывает дескрипторы, деля изображение на n частей по вертикали и по горизонтали
в функции pyramid Сначала считается HOG для всего изображения.
Потом изображение делится на 4, и считается  HOG для каждого подизображения

4. Цветовые признаки реализованы в классе HOGComputer, в методе computeCol
этот метод вызывается в методе HOGComputer::compute, который в свою очередь вызывается в методе ComputeHog класса HOG.
метод HOGComputer::compute высчитвает сначала hog подизображения, которое ему передает HOG::ComputeHog
А потом считает цветовые признаки для этого же подизображения, и конкетинирует их к вектору hog
